Algoritmo en python que permite la detección de movimiento utilizando la cámara
del computador, si detecta movimiento, captura y envía una fotografía por correo 
electrónico.

Para que funcione, se debe configurar el correo de gmail, en el archivo correo.py.


Posterior a ello, se debe ejecutar el archivo main.py y decir "Activar el modo
vigilancia" para la ejecución correcta de este.
